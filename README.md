# Cosmic Tic Tac Toe (React + Vite)

A space-themed, neon Tic Tac Toe game with:
- Human vs AI (minimax, with slight randomness among best moves)
- Choose X or O
- Score tracker (wins / losses / draws)
- Winning line highlight
- Optional sound effects (Web Audio API, no external assets)

## Run locally

```bash
npm install
npm run dev
```

Then open the URL Vite prints (usually `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```
